
# Simulación de una variable aleatoria Binomial

Este ejemplo implementa la simulación de la distribución Binomial
mediante el método de la pseudoinversa de la función de distribución
acumulada (CDF), como parte del producto integrador de la unidad
*Modelos probabilísticos aplicados*.

## Archivos

- `simulacion_binomial.ipynb`: cuaderno de Jupyter con la explicación
  y la implementación paso a paso.
- `simulacion_binomial.py`: script en Python con las funciones de
  simulación basadas en la pseudoinversa.

## Uso rápido

1. Abrir `simulacion_binomial.ipynb` en Jupyter.
2. Ejecutar las celdas para generar muestras de `X ~ Bin(n, p)` y
   comparar las frecuencias simuladas con las probabilidades teóricas.
